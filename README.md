# 🎓 Smart Campus Information System

A comprehensive Flask web application integrating **8 Python programming concepts** from the original CLI programs into a unified, responsive web platform.

## ✨ Features

| Module | Original Concept | Web Implementation |
|--------|-----------------|-------------------|
| Q1 | Student Registration & Grade Evaluation | Form input + conditional grading logic |
| Q2 | Course Enrollment Management | Dictionary-based course tracking |
| Q3 | Student Record Management | Lists, dicts, sets with set operations |
| Q4 | Sorting & Searching | Bubble sort + Linear/Binary search |
| Q5 | Fee Calculation | Functions with default parameters |
| Q6 | File Handling | Read/write CSV-style records + reports |
| Q7 | Directory Scanning | OS module + custom exception handling |
| Q8 | Performance Analytics | NumPy/Pandas/Matplotlib charts |
